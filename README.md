# 🎓 Django Student Management System
A simple Django CRUD web application to manage student records.

## ✨ Features
- Add Student
- View Students
- Edit Student
- Delete Student
- Django Admin Panel

## 🛠️ Technologies
- Python
- Django
- HTML

- Bootstrap

## 🚀 Installation
1. Clone the repository
```bash
git clone https://github.com/SONAL-THAKUR163/django-student-management.git
```
2. Open the project
```bash
cd django-student-management
```
3. Install Django
```bash
pip install django
```
4. Apply migrations
```bash
python manage.py migrate
```
5. Run the server
```bash
python manage.py runserver
```
6. Open in your browser
- Home: `http://127.0.0.1:8000/`
- Admin: `http://127.0.0.1:8000/admin/`
