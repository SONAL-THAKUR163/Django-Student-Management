# 🎓 Django Student Management System
A simple Django CRUD web application for managing student records. It allows users to add, view, edit, and delete student information through a clean web interface.

## 🌐 Live Demo
🔗 https://django-student-management-3.onrender.com

## ✨ Features
- ➕ Add Student Records
- 📋 View All Students
- ✏️ Edit Student Details
- 🗑️ Delete Student Records
- 💾 Store Data Using SQLite Database
- 🔐 Django Admin Panel
- 🌐 User-Friendly Web Interface

## 🚀 Installation

### 1. Clone the repository
```bash
git clone https://github.com/SONAL-THAKUR163/django-student-management.git
```
### 2. Open the project
```bash
cd django-student-management
```
### 3. Install Django
```bash
pip install django
```
### 4. Apply migrations
```bash
python manage.py migrate
```
### 5. Run the development server
```bash
python manage.py runserver
```
### 6. Open in your browser
- Home: `http://127.0.0.1:8000/`
- Admin: `http://127.0.0.1:8000/admin/`
